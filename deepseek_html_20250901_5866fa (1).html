Come aprire un file HTML nel browser

Una guida semplice per visualizzare le tue pagine web

Metodi per aprire un file HTML

1

Trovare il file sul computer

Individua la cartella in cui hai salvato il tuo file HTML. I file HTML hanno estensione .html o .htm.

2

Aprire con doppio clic

Fai doppio clic sul file HTML. Si aprirà automaticamente con il browser predefinito del tuo sistema.

Suggerimento: Se il file si apre con un programma diverso dal browser, continua a leggere per scoprire come cambiare questo comportamento.
3

Aprire tramite menu del browser

Puoi aprire il file HTML anche da dentro il browser:

  • Apri il browser (Chrome, Firefox, Edge, ecc.)
  • Premi Ctrl+O (o Cmd+O su Mac)
  • Scegli il file HTML che vuoi aprire
  • Clicca "Apri"
4

Trascinamento

Alcuni browser supportano il trascinamento: trascina semplicemente il file HTML dalla cartella direttamente nel browser.

Impostare il browser predefinito

Se i tuoi file HTML non si aprono con il browser che preferisci, puoi cambiare le impostazioni:

1

Windows

Vai in Impostazioni > App > App predefinite. Cerca "Scegli app predefinite per tipo di file" e associa .html al browser che preferisci.

2

Mac

Seleziona il file HTML, poi dal menu File scegli "Ottieni informazioni". Nella sezione "Apri con", scegli il browser desiderato e clicca "Cambia tutto...".

3

Linux

Dipende dalla distribuzione, ma generalmente puoi farlo dalle impostazioni di sistema cercando "Applicazioni predefinite" o "Preferenze sulle applicazioni".

Browser supportati

Puoi aprire file HTML con qualsiasi browser web moderno:

Chrome
Firefox
Edge
Safari
Opera
Nota: Il rendering della pagina potrebbe variare leggermente tra browser diversi, soprattutto per elementi CSS avanzati.

Risoluzione dei problemi

1

Il file non si apre correttamente

Assicurati che il file abbia estensione .html e che non sia danneggiato. Controlla anche di aver salvato le modifiche prima di aprire il file.

2

La pagina appare vuota o non formattata

Apri gli strumenti di sviluppo del browser (F12) e controlla la console per eventuali errori. Potrebbero esserci problemi nel codice HTML, CSS o JavaScript.

3

Le immagini non vengono caricate

Controlla i percorsi delle immagini nel tuo codice HTML. Se usi percorsi relativi, assicurati che le immagini siano nella posizione corretta rispetto al file HTML.
